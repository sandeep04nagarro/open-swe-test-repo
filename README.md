# open-swe-test-repo

This is a test repository for demonstrating Open SWE capabilities.

## Overview

Open SWE is an open-source coding agent built on LangGraph and Deep Agents. This repository serves as a testbed for exploring and validating Open SWE's functionality.

## Features

- **LangGraph Integration**: Built on the LangGraph framework for robust agent orchestration
- **GitHub Automation**: Full support for GitHub operations including issues, pull requests, and code reviews
- **Multi-tool Support**: Integrates with Linear, Slack, and various development tools
- **Sandboxed Execution**: Safe code execution in isolated environments

## Getting Started

### Prerequisites

- Python 3.9+
- Git
- GitHub CLI (`gh`) for GitHub operations

### Installation

Clone the repository:

```bash
git clone https://github.com/sandeep04nagarro/open-swe-test-repo.git
cd open-swe-test-repo
```

## Usage

This repository is intentionally minimal and acts as a sandbox for Open SWE to exercise its end-to-end flows. Typical workflows include:

- Creating and managing GitHub issues and pull requests
- Reading repository context and proposing minimal, focused changes
- Pushing branches, opening draft PRs, and posting review comments
